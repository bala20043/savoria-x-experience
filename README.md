# Savoria X — Premium Dining Experience

A cinematic luxury dining website built with React, TanStack Start, and Supabase.

## 🚀 Deployment Features
- **SSR Ready**: Configured for high-performance Vercel deployment.
- **Database**: Fully integrated with Supabase (Reservations & Orders).
- **Security**: Row Level Security (RLS) enabled for public transactions.
- **Aesthetics**: Premium Framer Motion animations and glassmorphism design.

## 🛠️ Vercel Setup
1. **Output Directory**: `.output`
2. **Environment Variables**:
   - `NITRO_PRESET=vercel`
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
